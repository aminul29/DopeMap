# DopeMap for Elementor

Interactive world map Elementor widget using jVectorMap.

## Features

- Custom world map style controls.
- Country-based map markers.
- Popup content per marker: title, subtitle, image, and button.
- Hover popup with click fallback for touch devices.

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Elementor plugin

## Widget

- Name: `Dope World Map`
- Category: `Dope Plugins`
